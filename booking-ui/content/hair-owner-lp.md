# 予約管理のストレスをゼロへ。

「予約の電話が鳴るたびに手が止まる」「空き枠の管理がぐちゃぐちゃ」「InstagramとLINEの予約がバラバラ」  
そんな毎日の小さなストレスが、気づけば大きな”機会損失”になっています。

**Kazuki Booking** は、  
LINE1つで「予約受付〜空き枠管理〜リマインド」まで自動化する、  
美容室オーナーのためのオンライン予約システムです。

---

## 導入前と導入後の違い

### 導入前（よくある悩み）

- 予約の電話対応で施術が中断される  
- 紙台帳やスプレッドシートでの管理で記入ミスが不安  
- ダブルブッキングや空き枠の反映漏れが起きがち  
- SNSやホットペッパーなど、予約経路がバラバラで把握しづらい  
- キャンセルや変更連絡のたびに手作業で調整している  

### 導入後（Kazuki Booking 導入後の世界）

- 予約は**すべてLINEで自動受付** → 施術に集中できる  
- 空き枠はカレンダーと自動連携し、**ダブルブッキングを自動で防止**  
- お客様はスマホから24時間いつでも予約OK  
- 予約確認・リマインド・キャンセル受付も自動化  
- 予約履歴が貯まるので、「来店周期」「来店回数」が一目でわかる  

---

## Kazuki Booking でできること

### 1. LINE連携の予約フォーム

- お客様は「LINE追加 → メニューと日時を選ぶ」だけ  
- 予約完了メッセージには、日時・メニュー・担当者を自動で送信  
- LINEのトーク履歴がそのままカルテ代わりに  

### 2. ダブルブッキング防止のスロット管理

- 予約枠ごとに ID を振り、**Durable Object で二重予約をロック**  
- 予約完了と同時に、その枠は自動的に「埋まり」に変更  
- Web予約とLINE予約が同じ在庫を見ているから、チグハグにならない  

### 3. リピート率を上げる自動フォロー

- 予約履歴から「前回来店から◯日経過したお客様」を自動で抽出  
- LINE で「そろそろカラーの時期ですね」などのフォローメッセージを送信可能  
- 放置されがちな既存客へのアプローチが、**ほぼノーメンテで回り続ける**

## 導入の流れ

1. **無料オンライン相談**  
   サロンのメニュー構成・席数・スタッフ人数をヒアリングします。

2. **初期設定（最短1〜2日）**  
   メニュー・施術時間・営業時間をもとに、予約枠のロジックを構築。  
   LINE公式アカウントと Kazuki Booking を接続します。

3. **テスト予約 & スタッフ共有**  
   オーナー様と一緒に、実際の予約〜確認までをテスト。  
   現場の運用とズレがないかを一緒に確認します。

4. **本番運用スタート**  
   SNSや店内POPから「LINE予約」へ誘導。  
   予約が入るたびに、オーナー様・スタッフ・お客様全員がLINEで同じ情報を共有できます。

## よくある質問

**Q. ITが苦手でも使えますか？**  
A. ほとんどの操作は「LINEのトーク画面」で完結します。  
   管理画面の初期設定も、オンラインで一緒に行うのでご安心ください。

**Q. 既存の予約システムから乗り換えられますか？**  
A. はい。現在の運用（紙 / Googleカレンダー / 他社システム）をヒアリングして、  
   極力ストレスなく移行できるように設計します。

**Q. 料金はいくらぐらいですか？**  
A. 席数・店舗数・メニュー構成によって変わりますが、  
   目安としては「電話対応が減ることで浮く人件費」で十分ペイできる価格帯を想定しています。  
   まずは無料相談で、お店に合うプランをご提案します。
